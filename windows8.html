Milestone: Windows 8


  • Microsoft released Windows 8 in 2012 which brought changes to the Windows operating system. It introduced a touch centric interface that worked across devices like desktops, laptops, tablets and smartphones. This interface known as Metro UI showcased tiles displaying real time information and granting access to favorite apps. Functionally Windows 8 brought improvements. It had good boot times, enhanced security features, and better performance compared to its previous versions. Additionally it introduced an app store where users could directly download and install new applications from the Microsoft Store. Windows 8 had a modern appearance with design elements and vibrant colors. The traditional Start button was replaced with a Start screen featuring grids of tiles based on user preferences. The operating system also included an upgraded task manager, file explorer, and improved support for monitor setups. While some users welcomed the updated touch oriented interface and enhanced performance there were others who struggled to adjust to the changes, from versions of Windows.
